Download like:

curl https://data.nodc.noaa.gov/ncei/ocads/data/0160558/MPI_SOM-FFN_v2020/spco2_MPI-SOM_FFN_v2020.nc --output spco2_MPI-SOM_FFN_v2020.nc

Use NCO to convert this variable name so they can be opened. Ie.  #However to run this, need NCO so either download or module load nco
ncrename -v date,t spco2_MPI-SOM_FFN_v2020.nc
